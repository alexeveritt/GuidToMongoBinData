Convert Guids to MongoDb BinData Objects


This simple tool makes it easy to convert Guids in to theBinData objects that are used by MongoDb for find queries.

All you need to do is enter a guid, select the encoding and press convert.

To save time when creating queries you can add a the name of the field that contains the guid e.g. _id and a full query will be generated that you can just paste straight into tools like MongoVue or RoboMongo.

Have a play, enjoy!
